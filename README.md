# Breaking Bad Theme

A collection of VS Code themes inspired by the Blue Sky lab and Los Pollos Hermanos from Breaking Bad.

## Available Themes

*   **Breaking Bad: Heisenberg** - Dark green and blue tones inspired by the Blue Sky lab and Walter White's persona.
*   **Breaking Bad: Los Pollos** - Vibrant red and yellow colors inspired by the Los Pollos Hermanos restaurant.

## Installation

1.  Open Chrome or your browser of choice.
2.  Download the `.vsix` file.
3.  Open VS Code.
4.  Go to Extensions -> "..." -> Install from VSIX...
5.  Select the downloaded file.
6.  Go to File -> Preferences -> Color Theme and select one of the Breaking Bad themes.
